# Dvara CRM Demo Script

## Prep
- Use an Admin account with a branch assigned.
- Ensure Storage rules are deployed (for attachment uploads).

## 1) Landing and Login
- Start on `/` and highlight the Dvara hero narrative.
- Click the CTA to reach `/login`.
- Sign in with email/password.

## 2) First-time Setup
- If prompted, complete profile setup (role + branch).
- Confirm the sidebar sections appear and breadcrumbs update.

## 3) Primary People + Household Context
- Go to **People** and create a new primary person.
- Fill household context (earning source, seasonality).
- Save and open the person detail page.

## 4) Products and Automation
- Add a **Loan** product with status **Active**.
- Show the auto-created task: "Business / income review".
- Edit the product to **Closed** and show the "Savings / pension conversation" task.

## 5) Interactions + Follow-ups
- Log an interaction with outcome **Follow-up Required** and a next action date.
- Show the auto-created follow-up task.

## 6) Risk Flags
- Edit the person and add a risk flag.
- Show the "At Risk" badge and the suggested "Insurance discussion" task.

## 7) Tasks Workflow
- Open **Tasks**, filter by status, and mark a task done.
- Show the "Next up" queue update.

## 8) Standard CRM Modules
- Create an opportunity, a meeting, and a phone call.
- Upload an attachment in **RFPs** or **Invoices**.
- Show the attachment link in the list view.

## 9) Reports
- Visit **Reports** and highlight live counts and officer activity summary.

## 10) User Management (Admin)
- Open **Settings -> Users**.
- Edit a user's role or branch assignment.

## 11) Deletion Flow
- From any list, delete a record via the action buttons.
- Confirm the row is removed from the list.
